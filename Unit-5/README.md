## Unit-5

Loop statement is important for coding. You can use it to reduce numerous, repeated or similar steps.

Loop statements envolve
 
* For loop
```c
for(初始值設定;條件運算式;更新運算式)
{
  條件運算式為真時，執行的主體敘述;
}
```

* While loop
```c
while(條件運算式)
{
 條件運算式為真時，執行的主體敘述;
}
```

* Do...while loop
```c
do
{
 執行敘述;
}while(條件運算式);
```
