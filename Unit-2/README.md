## Unit-2
In this unit, I will show you how to input and output data by using printf and scanf.
 * printf() and scanf()
```C
#include<stdio.h>
#include<stdlib.h>

int main()
{
  ... //設定資料型態
  printf("...輸出內容與格式特定字...",...變數或常數...);
  scanf("...格式特定字...",變數的位址);
  system("pause");
  return 0;
 }
```
